A Calculator class for performing quantum mechanics calculations for an entire system using Psi4.

  • functional (str) – The DFT functional to use for the Psi4 calculations.

  • quadrature_radial (int | None) – The number of radial points to use in the Lebedev quadrature implemented in Psi4.

  • scf_type (str | None) – The type of SCF to perform in Psi4.

  • read_guess (bool | None) – Determine whether or not read in wavefunction from previous calculation.

  • reference_energy (float | int | None) – The base potential energy for QM energies.
